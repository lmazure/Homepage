rpm

install
  • install a package
    #> rpm -i foobar.rpm [ toto.rpm ... ]
  • install a package only if a previous version is already present
    #> rpm -F foobar.rpm [ toto.rpm ... ]
uninstall
  • uninstall the foobar package
    #> rpm -e foobar
extract a file
  • extract a file from a RPM
    #> rpm2cpio foobar.rpm | cpio -iumd thefile
query on a package
  • list the capabilities provided by a package
    #> rpm -q --provides foobar
  • list the packages on which this package depends
    #> rpm -q --requires foobar
  • display the package description
    #> rpm -qi foobar
  • list the files present in the package
    #> rpm -ql foobar
  • list the files with their lengths, owner groups...
    #> rpm -ql --dump foobar
  • the -p flag allows to query an uninstalled package, for example:
    • display the uninstalled RPM package description
      #> rpm -qpi foobar.rpm
    • list the files present in the uninstalled RPM package
      #> rpm -qpl foobar.rpm
query on the machine
  • return the list of all the installed packages
    #> rpm -qa
  • indicate from which package is a given file
    #> rpm -qf /usr/include/string.h
