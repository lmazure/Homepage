These notes are very old (circa 1992), some are obsolete, some have been revised since.

cmd
 
cmd command interpreter
/C string execute the command specified by string and exist
/K string execute the command specified by string and remain

relative path
. current directory
.. parent directory
... parent of parent directory
.... parent of parent of parent directory

 
dir /p list the directory in page mode
dir /S recursive listing
dir /w list the directory in multicolumn mode
dir equivalent to dir *.*
dir name equivalent to dir name.*
dir .suf equivalent to dir *.suf
dir /b
 bare format
dir /b > list.txt record the content of a directory in a file (list.txt will appear itself in the list)
dir /od
 sort by increasing date
dir /on
 sort by name
dir /oe
 sort by extension
dir /og
 group directories first
dir /os
 sort by increasing size
-
 reverse sort order
dir /o-d
 sort by decreasing date
dir /a<attributes>
 display files with specified attributes
D
 directories
H
 hidden files
S
 system files
R
 read-only files
A
 files ready for archiving
-
 prefix to negate an attribute


 
echo
 print a message

 
exit exit the batch script

 
doskey start DosKey

start Window applications
start . display a window of the content of the current library
start tips.txt display tips.txt in a Windows application

 
del /p
erase /p		 ask for a confirmation of file deletion

 
date display the date and ask for a new value
date dd.mm.aa set the date
time display the time and ask for a new value
time hh[:mm[:ss:[.cc]]] set the time

 
cls clear the screen

 
cd
chdir		 (without argument) return the name of the current directory

 
ver return the DOS version

 
type list a file on the screen
more display a file sreen per screen

 
rename
ren		 rename a file

 
sort sort the lines (case independant)
/t reverse the sort order
/+n sort on the nth column

 
verify on DOS verifies that file are correctly recorded
verify off DOS performs no verification
verify display the current mode

 
vol return the volume name of a unit
label define the label of a volume

 
tree graphicaly list a directory content
/f display the name of the file in each directory
/a use graphic characters to dsplay more quickly

 
path display the current path
path unit:path;unit:path;...;unit:path
set path=unit:path;unit:path;...;unit:path		 define the path
set path=unit:path;%path% complete the path
path;
set path=		 define an empty path

 
prompt define the DOS prompt
$q character =
$$ character $
$t current hour
$d current date
$p current directory of default unit
$v version number
$n default unit
$g character >
$l character <
$b character |
$- line return
$e character Esc
$h backspace

 
set list the environment variables
set var= earse an environment variable
set var=string define an environment variable

 
fdisk configure a hard drive

 
find "string" file search a string in a file
/v display all the lines not containing the string
/c only return the number of lines
/n prepend each line by its number in the file

 
mem display the free and used memory
/c
/classify		 list programs present in memory
/d
/debug		 list programs and internal controllers present in memory
/f
/free		 display the amount of free memory left
/p
/page		 pause after each screen

 
keyb fr configure the keyboard to French

 
fc compare two files
/a abvreviated ASCII comparison, only the first and last lines of zones of difference are displayed
/b binary comparison
/c case independent comparison
/l ASCII comparison
/lb num define the internal buffer size in number of lines
/n display the line numbers
/t does not transform tabs into spaces
/w compress spaces and tabulations
/nnnn maximum number of identical lines, but inserted between different lines, that will be considered different by extension
comp compare two files

 
break on disk reads/writes can be interrupted by Ctrl-C
break off disk reads/writes cannot be interrupted
break return the current switch state

 
chkdsk return a disk check report
/f in case of a problem, ask for a reparation by creating a file containing the lost chain
/v display each file name while it is checked

 
replace update anterior versions of files
/a add the new files to the target directory instead of replacing existing files
/p ask for a confirmation before replacing a file
/r replace also the write protected files
/s searchin the subdirectories of the target directory (there is never a search in the subdirectories of the source directory), incompatible with /a
/u replace only the files which are older in the target directory than in the source directory
/w prompt before strating the command

 
graftabl allow the display, in graphic mode, of an extended character set (CGA)
graftabl xxx use the xxx page
graftabl ? display a help message
graftabl /sta display the current status

 
diskcopy copy floppies (the files are not defragmented)
/1 copy a single face

 
backup backup files
/s save the subdirectories
/m save only the files modified since last backup
/a add only the files which does not exist in the backup
/f format the floppy with a given capacity
/d save only the files modified since a given date
/t save only the files modified since a given hour
/l create an entry in a given journal file
restore restore files
/s restore the subdirectories
/p ask for a confirmation for file which are write protected or modified since the backup
/b restore only the files modified before a given date
/a restore only the files modified after a given date
/e restore only the files modified before a given hour
/l restore only the files modified after a given hour
/m restore only the files modified since the last backup
/n restore only the files which does not exist anymore
