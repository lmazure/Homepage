• Windows Script Host:
    wscript.exe provides a windows-based property sheet for setting script properties
    cscript.exe provides command line switches for setting script properties
    .vbs extension indicates a VBScript
    .js extension indicates a JScript
When modifying a Visual Basic Script (.vbs file) properties with
  • cscript
  • wscript
  • a right click on the script icon
a Windows Script Host Settings file (.wsh file) file is created recording the properties.
The script can be launched by double clicking the .wsh icon.

Trivial example of a VB Script:
The file test.vbs contains
MsgBox "Hello, world!"
double click on its icon to run it or execute "wscript test.vbs" or "cscript test.vbs"

  • get a DOS window:
    • Start > Run and use the cmd program
     
  • change an environment variable:
    • Start > Settings > Control Panel > System > Advanced > Environment variables
     
  • to redefine the partitions of a hard disk
    execute compmgmt.msc
    select Disk Management

  • plug/unplug a device:
    • Start>Settings>Control Panel>Add/Remove Hardware
     
  • attach a keyboard shortcut to a program shortcut on the desktop or in the start menu:
    • right click on the shortcut
    select Properties > Shortcut > Shortcut key
    choose a letter
    the program will now be started by the Ctrl+Alt+letter shortcut
    warning: these shortcuts have priority on the application shortcuts: Ctrl+Alt+N will not work anymore in Word for example.
     
  • the shortcuts available in QuickLaunch are stored in:
    C:\Documents and Settings\mazurla\Application Data\Microsoft\Internet Explorer\Quick Launch

  • display/undisplay the "Docs/Mes Documents" folder on the desktop (seems to not work on Windows XP):
    In the Explorer Window: Tools>Folder Options>View>Show My Documents on the Desktop

  • enable/disable LAN connection:
    • Start>Settings>Control Panel>Network and Dial-up Connections>Local Area Network
     
  • create a new connection
    Win95:
    Démarrer>Programmes>Accessoires>Accès réseau à distance>Nouvelle connexion
    Win2000:
    Start>Settings>Network and Dial-up Connections>Make New Connection
